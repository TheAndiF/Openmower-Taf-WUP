BL_Add WAHA session auto repair
Date: 2026-06-27
Summary:
- Added configurable WAHA session selection with WAHA_SESSION.
- Added automatic WAHA session self-healing: start STOPPED sessions and restart FAILED/CRASHED or stuck STARTING sessions.
- Added watchdog and send-path readiness checks with cooldown, wait timeout and maximum restarts per hour.
- Published retained diagnostics under messenger/waha/session/repair/# and added manual MQTT actions for session start/restart.
- Added WHATSAPP_START_SESSION to the WAHA compose example so the configured session can start when WAHA starts.
- Updated GPS status coordinate extraction to prefer valid world_pose WGS84 coordinates for Google Maps and never use local pose.x/y.
- Documented the new environment variables, MQTT topics and operational checks.
